Everything you need to know about Lokei.

Getting Started
Lokei is a private AI assistant that lives in your Mac's menu bar. It connects to Ollama — a free, open-source local AI runtime — to run AI models entirely on your own hardware. No internet required, no subscription, no data leaving your Mac.
You need two things: (1) A Mac running macOS 13 Ventura or later, on either Intel or Apple Silicon. (2) Ollama installed and running — it's free and takes about two minutes to set up. Lokei handles the rest.
Lokei works on any Mac running macOS 13 Ventura or later, including both Intel and Apple Silicon (M1/M2/M3/M4) Macs. If you're on macOS 12 or earlier, you'll need to update your operating system first.
Four steps: (1) Download and install Ollama from ollama.com. (2) Purchase and install Lokei from the Mac App Store. (3) Open Lokei, go to Settings → Models, and download an AI model — no Terminal needed. (4) Click the leaf icon in your menu bar and start chatting. Lokei walks you through all of this on first launch.

AI Models
It depends on your Mac's RAM. For 8GB RAM Macs, we recommend Qwen 2.5 Coder (3B) for coding or Llama 3.2 (3B) for general use — both are fast and run well. For 16GB+ RAM, you can run larger models like DeepSeek R1 (7B) or Mistral (7B) for more capable responses. Lokei's Model Library shows RAM requirements for every model.
No. Lokei includes a built-in Model Library where you can browse and download models with a single click — no Terminal required. Models are rated by RAM requirement so you know exactly what will run well on your Mac.
Small models (1-3B parameters) are typically 1–2 GB. Medium models (7B) are around 4–5 GB. Lokei's Model Library shows the exact download size for each model before you download it. Models are stored in Ollama's directory, not inside the Lokei app itself.
Yes. Lokei automatically detects all models installed through Ollama and shows them in the model picker. If you've been using Ollama before Lokei, all your existing models are immediately available.
Features & Usage
Click the Lokei leaf icon in your menu bar, or press ⌥ L (Option + L) from anywhere on your Mac to instantly toggle Lokei open and closed. The shortcut requires accessibility permissions — Lokei will walk you through enabling it the first time you use it.
Specialist Modes (called Presets in the app) are different system prompts that tune Lokei's personality and behavior for specific tasks. Built-in modes include General, Code Helper, Debug Mode, Explain It, Writing, and Brainstorm. You can also create unlimited custom modes in Settings → Presets.
Project Context lets you tell Lokei about your project once — your tech stack, coding style, architecture decisions — and Lokei quietly includes that knowledge with every message you send. No more repeating yourself every conversation. Set it up in Settings → Context.
Yes. You can drag and drop files onto Lokei's chat window, click the file icon in the input bar, or paste clipboard content. Lokei supports code files, text files, PDFs, and more. Large files are automatically truncated to keep responses fast.
Yes. Click the export icon in the header to save your current conversation as a Markdown file. This works in both the menu bar popover and the expanded full window.

Troubleshooting
Open your Applications folder and launch the Ollama app, or run ollama serve in Terminal. Once Ollama is running, click "Try Again" in Lokei. If you haven't installed Ollama yet, download it from ollama.com.
Switch to a smaller model — 3B parameter models are significantly faster than 7B models, especially on 8GB RAM Macs. Also, long conversations slow down over time because the model processes all previous messages. Use "Summarize & Compress" in the chat (appears after 20 messages) or start a new chat to speed things back up.
The ⌥L global shortcut requires accessibility permissions. Go to System Settings → Privacy & Security → Accessibility, and make sure Lokei is enabled. If Lokei isn't listed, try pressing ⌥L once — Lokei will prompt you to grant access.
