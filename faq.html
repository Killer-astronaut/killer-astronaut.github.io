Getting Started

Lokei is a Mac app that puts a private AI assistant in your menu bar — the row of icons at the top right of your screen. Click the leaf icon and you have instant access to an AI that can answer questions, help with code, improve your writing, brainstorm ideas, and more.

The key difference from ChatGPT or Claude: everything runs directly on your Mac. No internet required. No data sent anywhere. Completely private.

Two things — both free:

  1. Ollama — a free app that runs AI models on your Mac. Download it at ollama.com. Just download and open it like any Mac app.
  2. An AI model — think of this as the AI brain. Lokei's built-in Model Library lets you browse and download models with one click — no technical knowledge needed.

Once both are installed, Lokei finds everything automatically. The whole setup takes about 5 minutes.

Absolutely — Lokei was designed with non-technical users in mind. The built-in setup guide walks you through every step. The Model Library lets you download AI models with a single click, just like downloading an app.

If you've ever used ChatGPT or Siri, you already know how to use Lokei. The only difference is everything stays on your Mac.

Tip
If you ever get stuck, open Lokei → Settings → About → Show Setup Guide to replay the full walkthrough.

Lokei requires macOS 13.0 (Ventura) or later. It works on both Intel and Apple Silicon Macs.

Performance depends on your Mac's RAM. Macs with 8GB RAM work great with smaller models (1-3GB). Macs with 16GB+ RAM can run larger, more powerful models with faster responses. Lokei's Model Library clearly labels which models work best for your hardware.

Installation
  1. Go to ollama.com and click Download for Mac
  2. Open the downloaded file and drag Ollama to your Applications folder
  3. Open Ollama from your Applications folder — a small icon will appear in your menu bar
  4. That's it — Ollama is now running in the background
Tip
Ollama needs to be running whenever you use Lokei. If Lokei shows an error, check that the Ollama icon is in your menu bar.

The easiest way is directly inside Lokei:

  1. Open Lokei and click the gear icon to open Settings
  2. Click the Models tab
  3. Browse the model library — each model shows its size and RAM requirements
  4. Click Download next to any model — a progress bar shows the download
  5. Once complete the model appears in your dropdown and is ready to use

Using Lokei

Lokei comes with six built-in modes that change how the AI responds:

  • General — Friendly all-purpose assistant for everyday questions
  • Code Helper — Writes clean, commented code and explains what it does
  • Debug Mode — Finds bugs fast and tells you exactly how to fix them
  • Explain It — Simplifies complex topics using plain language and analogies
  • Writing — Improves grammar, clarity and tone while keeping your voice
  • Brainstorm — Generates creative ideas and builds on your thinking

You can also create your own custom modes in Settings → Presets.

The clipboard button (the document icon at the bottom left) lets you instantly send whatever is on your clipboard to Lokei for analysis.

For example: copy some code from your editor, click the clipboard button in Lokei, and it will automatically ask what you'd like help with — reviewing it, finding bugs, explaining what it does, or suggesting improvements.

It works with any text — code, an email draft, a paragraph you want to improve, a problem you want to think through.

Project Context lets you give Lokei background knowledge about your work — once — so you never have to repeat yourself.

For example, you can create a context called "My App" and write: "This is a Swift iOS app using SwiftUI and MVVM. The app is a to-do list manager targeting iOS 17." Once activated, Lokei includes this context with every message, so it always understands what you're working on.

To set it up: Settings → Context → New Context. You can create multiple contexts for different projects and switch between them anytime.

Pro tip
Keep your context under 500 words for best performance on smaller models.
